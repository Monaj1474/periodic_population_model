# periodic_population_model
Minitab model design first version;

In this Research, a set of annual population data for Halibut (a kind of Fish) is selected from GPDD of Center for population biology. After obtaining the first phase and the period of the periodic cycle, a highly accurate periodic sinusoid statistical model has been derived to fit on the dataset with R^2=98% , in order to predict the future values and next cycles.
